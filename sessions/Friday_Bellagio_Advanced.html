LFS303 - AI/ML in Life Sciences: Predictive Modeling with Amazon SageMaker In this workshop, learn how to use Amazon SageMaker and other AI/ML services on AWS to build predictive data models for life sciences. Learn how to incorporate diverse data types, ranging from sensor data to imagery, into machine learning models. At the end of this workshop, you should be able to deploy these models to monitor real-world scenarios in life sciences. We provide a step-by-step guide to complete the workshop. However, in order to take full advantage of this workshop, we recommend the following prerequisites: a laptop with Wi-Fi capability and an internet browser, an AWS account that has an IAM user with administrator permissions and configured to use the AWS CLI, which you can install here. Also, you should have basic familiarity with the following AWS services: Amazon SageMaker, AWS CloudFormation, Amazon S3, AWS Lambda, Amazon DynamoDB and Amazon Elasticsearch Service. Finally, you should have a basic understanding of Python. View Less Workshop Bryan coon
